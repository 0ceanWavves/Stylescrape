Website Cloner API Server

This is the API server for the Website Cloner tool. Access the GUI at http://localhost:3000 to use the tool.

Available API Endpoints

Endpoint Method Description
/api/clone POST Clone a website
/api/sites GET List cloned sites
/api/extract-design POST Extract design elements from a cloned site
/sites/... GET Access static files from cloned sites

Quick Usage

To use the Website Cloner tool:

  1. Start the backend server (this server)
  2. Start the frontend GUI (from the website-cloner-gui directory)
  3. Open your browser to http://localhost:3000
  4. Enter a website URL and options
  5. Click "Clone Website"
